assignment SFA-4029

Refactor Auth Middleware for Edge Runtime Compatibility

The current authentication middleware relies on Node.js specific APIs (Buffer, crypto) which are not available in the Vercel Edge Runtime. We need to transition to Web Standard APIs (SubtleCrypto, Uint8Array) to ensure the application can be deployed globally with low latency.

Requirements:
- Replace 'crypto' with 'globalThis.crypto'
- Ensure JWT verification works in middleware.ts
- Add unit tests for edge runtime environment.
