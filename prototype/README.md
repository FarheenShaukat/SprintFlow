# SprintFlow AI UI Prototype

This folder contains the UI screens extracted from `stitch_sprintflow_task_management (1).zip`.

Open `index.html` to launch the screen list, or open any screen directly:

- `landing/code.html`
- `login/code.html`
- `register/code.html`
- `dashboard/code.html`
- `workspace/code.html`
- `kanban/code.html`
- `task-detail/code.html`
- `reports/code.html`

Notes:

- The screens preserve the exported HTML/Tailwind UI from the zip.
- Only one Kanban screen is kept in the prototype.
- AI insights are represented through the reports and task detail screens, matching the project brief's no-duplicate-screens rule.
- The copied screens were mechanically rebranded from Zenith PM to SprintFlow AI.
